## Usage

**Before ending a session:**
```
Write your current progress to SCRATCHPAD.md using the template
in demo-files/08-context-management/scratchpad-template.md.
Include all sections.
```

**Starting the next session:**
```
Read SCRATCHPAD.md and the relevant spec. Continue from where
the previous session left off. Start with the first item in
"Remaining Work."
```
